Design system

What's changing, what's staying

🖋
Typography DM Serif Display for artist names. Inter for all UI text. Big, expressive headings on near-black backgrounds.
🎨
Per-artist color extraction Each artist gets a color derived from their image. Flows through genre tag, track strip, play button, save button.
examples
💜
Purple fallback accent (#8b5cf6) Used for nav active state, saved hearts, drawer save button, and any element where no artist color is loaded yet.
purple · dark purple · base black
🎵
Track strip (prominent) First track name shown inline. Stacked album art thumbnails. Tinted border + bg using artist color. Circular play button.
🪟
Tap-to-expand drawer Card image peeks behind. Drawer slides up with full track list, reason text, genre tags, and Spotify link.
🖼
Full-bleed editorial images No rounded card images — artist photos bleed edge to edge. Gradient overlay fades to card background.
📱
Nav: Feed · Saved · Settings Groups removed. Three clean destinations. Bottom tab on mobile, top bar on desktop.
🖤
Base palette Near-black (#080808) base. Card surfaces (#0f0f0f). Borders at 8–10% opacity. No navy — cleaner, more confident dark.
